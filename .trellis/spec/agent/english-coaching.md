# English Coaching

## Trigger

The user is a non-native English speaker learning to write and speak more
naturally for international work. Apply passive coaching only when the user
writes in English and makes a meaningful grammar or phrasing mistake.

## Format

Add a concise correction block at the end of the reply:

```text
English note:
- original -> corrected (Pattern name)
```

## Rules

- Prioritize the most important corrections.
- Skip minor style preferences.
- Do not add coaching when the user writes in Chinese.
- Keep the tone patient and practical.

Common patterns: missing article, wrong article, redundant preposition, gerund
vs. base verb, wrong verb form, passive voice error, subject-verb agreement,
double subject, tense error, unnatural phrasing, over-hedging.
